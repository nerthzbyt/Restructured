Indicadores propios

Todos los indicadores usan normalización Welford z-score con ventana mínima de 5 muestras antes de calibración completa.

IDNombrePesoCategoríaDescripción
pioPressure Imbalance Orderbook0.25MicroestructuraPresión bid/ask ponderada por distancia al mid (exp decay)
egmEnhanced Gravity Metric0.30MicroestructuraPIO + asimetría + bonus ROL. Filtro direccional principal
ildImplied Liquidity Depth-0.15LiquidezCoste notional para mover ±target_move. Peso negativo
rolRate of Liquidity0.10MicroestructuraΔ liquidez / Δt — absorción o retirada rápida
ogmOrder Gap Metric0.05MicroestructuraGaps grandes vs medianos en el libro
tfiTrade Flow Imbalance0.25FlujoDesequilibrio qty en últimos 10 trades
momMomentum Composite0.16MomentumEMA5/20 + retornos multi-timeframe

Señal Combined

combined = scale × (w_pio·pio_z + w_egm·egm_z + w_ild·ild_z
           + w_rol·rol_z + w_ogm·ogm_z + w_tfi·tfi_z + w_mom·mom_z)

# Defaults: scale=10, buy_th=6.0, sell_th=-6.0, hold_band=3.0
# Optimizable vía POST /agent/optimize (Optuna)

Niveles de predicción L0–L4

L0

Sin señal

0% — Sin calibración o data_ok=false. No operar.

L1

Observación

25% — Calibrado, |combined| < hold_band. Limit preferido.

L2

Direccional débil

50% — Señal parcial, veto EGM o ML < 0.55.

L3

Accionable

75% — Umbrales + EGM OK. Market+IOC validado.

L4

Alta convicción

92% — Composite ≥85.6, execution ≥90. Prioridad máxima.
